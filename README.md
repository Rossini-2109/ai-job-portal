# AI Job Portal

An AI-powered job portal that matches job seekers with recruiters using
resume parsing and TF-IDF based similarity.

## Features
- Job Seeker & Recruiter authentication
- Job posting & applications
- Resume upload & parsing
- AI-based job matching
- Email notifications

## Tech Stack
- Backend: Flask
- Database: MySQL
- AI: TF-IDF + Cosine Similarity
